## Sonar Software V2

### Chrome Extension to change some of the layout and style of your instance

**What does it do?**

- [x] Removes a lot od passing.
- [x] Redesigns Schedule que and about of space used for dispatch view
- [x] Adds a few minor features
- [x] and More

**How to install**

1. Download Zip file.
2. Unzip into a none folder
3. Open Chrome
4. in the url bar go to chrome://extensions
5. Top Right, toggle on Developer Mode
6. Click Load unpacked at the top right
7. Select Extension's folder.
8. Refresh your sonar instance.
